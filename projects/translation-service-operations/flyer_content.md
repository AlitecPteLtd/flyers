# Flyer Content — Translation Service Operations

## Cover / Top Section

**Title:** Translation Service Operations

**Subtitle:** Language Projects — From Quote to Delivery with Odoo

**Introduction:**
Alitec's translation solution helps language service providers manage client quotes, translation projects, vendor assignments, and billing in one Odoo platform — with third-party translation tool integration, document-level vendor POs, and language-pair product catalogs.

## Quick Benefits

1. **Third-Party Tools** — Connect your CAT or translation platform and sync document status and assignments.
2. **Vendor PO Automation** — Create purchase orders from documents with word, character, or minimum-charge pricing.
3. **Project Control** — Deadlines, project managers, turnaround times, and delivery formats on every job.
4. **Language-Aware Catalog** — Products mapped by source language, target language, and TR/TEP/REV service type.

## Key Capabilities

1. **Quotation Templates** — Pre-fill memo notes, turnaround time, and delivery format on new quotes.
2. **Sale to Project** — Auto-create projects with deadlines, PM, and analytic accounts from orders.
3. **CAT Tool Integration** — Connect third-party translation tools and pull document assignments and weighted counts.
4. **Translation Documents** — Track translators, reviewers, status, and weighted words or characters.
5. **Vendor Purchase Orders** — Generate POs from document assignments using supplier pricelists.
6. **Language-Pair Products** — Catalog products by source, target, and translation service type.
7. **Project File Folders** — Auto-copy folder structure when a new project starts.
8. **Job Order Reports** — Print quotations, invoices, and job orders with custom layouts.

## Business Workflow

1. **Quote Client** — Build quotation with language lines, deadline, and project details.
2. **Confirm & Kickoff** — Convert to project, launch the CAT job, and copy project folders.
3. **Assign Vendors** — Sync translators and reviewers from translation tool assignments.
4. **Create Vendor POs** — Generate purchase orders from weighted word or character counts.
5. **Invoice Client** — Bill completed work with branded quotation and invoice documents.

**Control modes:**
- Source and destination language validation
- TR / TEP / REV service type routing
- Word vs character unit pricing
- Minimum charge rules on vendor rates
- Analytic account per project

**Additional features:**
- Bulk contact and quotation import from Excel
- Sales team data security
- Delivery order reports from sales orders
- Third-party CAT tool status cron sync
- PM notification when CAT project is ready
