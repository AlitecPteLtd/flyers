## Scope

This flyer presents an Odoo solution for self-storage and storage rental operators. The source modules show customization around CRM storage enquiries, 2D unit inventory, unit type master data, rental contracts, customer portal products, payment/invoice visibility, recurring rental billing, deferred revenue, migration continuity, branch/company control, and commission tracking.

## Source Modules Referenced

- `bv_2d_inventory_view`: 2D storage unit map, branch/location/floor/section positioning, unit status colors, merged units, and rental order history.
- `bv_product_custom`: storage unit type details, size/volume/area, rent, deposit, location, and unit features.
- `bv_crm_custom`: storage enquiry capture, requested size range, move-in date, duration, aircon storage, unit type, branch, and conversion timing.
- `bv_rental_extended`: rental contracts, move-in/move-out handling, termination dates, notifications, paid-through information, and branch/location context.
- `bv_recurring_rental_extend`: recurring rental sequence, subscription plans, and invoice generation.
- `bv_portal_custom`: customer portal rental products, invoices, payments, paid-through dates, and outstanding invoice handling.
- `bv_deferred_revenue`: deferred revenue handling for rental income.
- `bv_company_migration`: migration continuity for rental data, old sale order linkage, company/branch movement, deposits, and invoices.

## Flyer Message

The flyer positions the solution as a connected operating platform for storage rental businesses: lead capture, unit matching, rental order creation, portal visibility, payment handling, recurring invoicing, and branch-level unit visibility.
