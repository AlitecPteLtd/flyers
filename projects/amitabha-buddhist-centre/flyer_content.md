# Flyer Content — Amitabha Buddhist Centre

## Cover

**Title:** Amitabha Buddhist Centre

**Subtitle:** Member Contacts, Personal Records & Website Migration with Odoo

**Introduction:**
This Amitabha Buddhist Centre customization focuses on trusted member and contact records, legacy database continuity, partner-grade descriptions, personal information fields, and website homepage migration.

## Quick Benefits

1. **Member Identity** — Member number, old database number, NRIC, first name, and last name fields.
2. **Personal Profile** — Birthdate, nationality, age range, address unit, gender, and language support.
3. **Legacy Continuity** — Keep old member numbers, last update date, last payment, and expiry dates visible.
4. **Website Migration** — Rebuild homepage content, program links, events snippets, and centre story in Odoo.

## Key Capabilities

1. **Member Number Control** — Track member number, old DB member number, NRIC, and identity fields in contacts.
2. **First / Last Name** — Support separate first-name and last-name capture while keeping Odoo name valid.
3. **Membership Dates** — Record last payment date, old last update date, and member expiry date.
4. **Address Unit Detail** — Add house/unit number visibility beside address fields for Singapore contact records.
5. **Member Grade Notes** — Add grade descriptions for clearer member category and partner-level management.
6. **Contact List Views** — Show member fields in list views for faster search, review, and data cleanup.
7. **Nationality & Personal Info** — Use OCA personal-information extensions for richer contact profile management.
8. **Website Homepage** — Migrate centre homepage, events, program links, and story sections into Odoo Website.

## Business Benefits

1. **Legacy Data Continuity** — Old member numbers and payment dates remain visible in Odoo.
2. **Cleaner Member Profiles** — Identity, address, nationality, and personal information stay structured.
3. **Safer Contact Creation** — Fallback name logic prevents missing-name contact insert failures.
4. **Website Continuity** — Homepage content, programs, events, and centre story move into Odoo.
