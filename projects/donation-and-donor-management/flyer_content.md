# Flyer Content — Donation and Donor Management

## Cover

**Title:** Donation and Donor Management

**Subtitle:** Donation Records, Donor Profiles & Website Migration with Odoo

**Introduction:**
This Odoo customization focuses on trusted donation and donor records, legacy database continuity, donor-grade descriptions, personal information fields, and website homepage migration.

## Quick Benefits

1. **Donor Identity** — Donor number, old database number, NRIC, first name, and last name fields.
2. **Donor Profile** — Birthdate, nationality, age range, address unit, gender, and language support.
3. **Donation Continuity** — Keep old donor numbers, last update date, last donation/payment, and expiry dates visible.
4. **Website Migration** — Rebuild homepage content, program links, events snippets, and centre story in Odoo.

## Key Capabilities

1. **Donor Number Control** — Track donor number, old DB donor number, NRIC, and identity fields in contacts.
2. **First / Last Name** — Support separate first-name and last-name capture while keeping Odoo name valid.
3. **Donation Dates** — Record last donation/payment date, old last update date, and donor expiry date.
4. **Address Unit Detail** — Add house/unit number visibility beside address fields for Singapore contact records.
5. **Donor Grade Notes** — Add grade descriptions for clearer donor category and partner-level management.
6. **Donor List Views** — Show donor fields in list views for faster search, review, and data cleanup.
7. **Nationality & Personal Info** — Use OCA personal-information extensions for richer contact profile management.
8. **Website Homepage** — Migrate homepage, events, program links, and story sections into Odoo Website.

## Business Benefits

1. **Legacy Donation Continuity** — Old donor numbers and payment dates remain visible in Odoo.
2. **Cleaner Donor Profiles** — Identity, address, nationality, and personal information stay structured.
3. **Safer Contact Creation** — Fallback name logic prevents missing-name contact insert failures.
4. **Website Continuity** — Homepage content, programs, events, and centre story move into Odoo.
