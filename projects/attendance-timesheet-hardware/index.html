Biometric capture, daily in/out processing, Excel payroll exports, and site-ready timesheet control

Alitec's attendance and timesheet solution on Odoo helps operations teams connect biometric punch data, employee attendance control, and team-based timesheet work in one managed workflow. It brings device sync, daily in/out processing, and export-ready timesheet control into one cleaner operation flow.

Direct device sync

Bring biometric logs into Odoo with device, area, and connection visibility.

Daily attendance control

Build one simple table with in/out time, status, and device traceability.

Shift-aware processing

Split day and night hours, preserve manual overrides, and review missing punches.

Timesheet-ready operations

Import worker sheets from Excel or CSV and move them through approval.

KEY CAPABILITIES

ZKTeco device registry

Track devices, area, timezone, connection state, last signal, user count, and attendance count inside Odoo.

Employee code mapping

Match device PIN or code combinations back to Odoo employees for reliable log ownership.

Attendance log capture

Download or sync raw punch logs, keep device references, and prepare the records for attendance processing.

Smart and punch processing

Convert logs into attendance using either shift-aware smart logic or punch-state rules with tolerance handling.

Simple attendance table

Store one daily record per employee with device in and out details, worked hours, and completion status.

Shift and team control

Split day and night hours, preserve manual overrides, and apply current or dated team history in output.

Excel export pack

Generate staff and worker attendance sheets with check-in, device name, NT, OT, and period-ready layouts.

Bulk timesheet import

Import site timesheets from Excel or CSV, auto-create workers, teams, and positions, then confirm and approve.
